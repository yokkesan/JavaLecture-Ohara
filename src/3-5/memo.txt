①-1 while文（前判定）

while (条件式) {
    繰り返す処理;
}


特徴
	•	条件を先に判定
	•	条件が false なら 1回も実行されない
	•	回数が事前に確定しない処理向き


①-2 do-while文（後判定）

do {
    繰り返す処理;
} while (条件式);

特徴
	•	処理を先に1回実行
	•	その後に条件判定
	•	必ず1回は実行される




for ( 初期化 ; 条件式 ; 更新 ) {
    繰り返し処理
}

for ( int i = 1 ; i <= 3 ; i++ ) {
      ↑初期化      ↑条件式      ↑更新
}
